## Features

* Programmatic UI (No Storyboards).
* Custom UIViews, custom TableViews for user interface.
* MVC Design Pattern.
* Custom UISearchBar.
* REST API.
* Network Calls.
* UserDefaults.
* Support for light & dark mode.
